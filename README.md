# FaceEarth

FaceEarth は、あなたの顔と似ている地球上の風景を探し出すメディアアートプロジェクトです。
自分の顔の特徴が、遠く離れた土地の風景とどのように共鳴するのかを可視化します。

## コンセプト

「顔」と「大地」。一見関係のない2つの存在を、AIの目を通して結びつけます。
顔の輪郭、パーツの配置、色合いなどを解析し、膨大な風景データの中から最も類似度の高い場所を特定。
さらに、顔のどの部分が風景のどの部分と対応しているのかを、美しいラインで結んで表現します。

## Philosophy

> 「土地を愛しているのか、果たして自分たちは？」

私たちは普段、土地に対してどれほどの愛着を持っているでしょうか？
何が人間と土地を結びつけるのでしょうか？住んだ年数？育った環境？

このプロジェクトは、**「共通点」**こそが親近感を生み、興味や愛を育むきっかけになるのではないかという仮説に基づいています。
AIによって強引にでも自分と土地の「共通点（類似性）」を見出すことで、遠く離れた見知らぬ土地に親近感を抱き、そこから環境問題や地球全体への興味・愛へと繋げていく。
そんな「占い」のような、あるいは「運命」のような体験を提供するメディアアートです。
（2024年 ニューヨークにて発表）

## 機能

- **顔類似度解析**: アップロードされた顔画像を解析し、似ている風景を探します。
- **マッチング可視化**: 顔のランドマーク（目、鼻、輪郭など）と、風景画像の対応するポイントを線で結び、視覚的なつながりを生成します。
- **位置情報**: 似ている風景の緯度・経度を表示します（現在はデモ用データ）。

## 技術スタック

### Backend (`/back`)
- **Python**: 主要言語
- **FastAPI**: Web APIフレームワーク
- **OpenCV**: 画像処理
- **Mediapipe**: 顔ランドマーク検出
- **CLIP (OpenAI)**: 画像の意味的類似度計算
- **PyTorch**: 深層学習フレームワーク

### Frontend (`/front`)
- **React**: UIライブラリ
- **Axios**: API通信

## 実行方法

### バックエンド

```bash
cd back
# 必要なライブラリをインストール
pip install -r requirements.txt
# サーバーを起動
uvicorn main:app --reload
```

サーバーは `http://127.0.0.1:8000` で起動します。

### フロントエンド

```bash
cd front
# 依存関係をインストール
npm install
# アプリケーションを起動
npm start
```

ブラウザで `http://localhost:3000` にアクセスしてください。

## 使い方

1. フロントエンド画面で「ファイルを選択」から自分の顔写真をアップロードします。
2. 「解析」ボタンを押すと、類似度と場所の座標が表示されます。
3. 「画像送信して結果を表示」ボタンを押すと、顔と風景が線で結ばれたアートワークが生成・表示されます。
